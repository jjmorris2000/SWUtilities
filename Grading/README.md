# CanvasExpand

CanvasExpand is a tool to expand Canvas project archives. To run from source, see the sample main method in ExpanderTools.

To run as a standalone application, use the jar file. It can be run as a commandline program as:
```
java -jar CanvasExpand.jar [input_archive] [output_directory]
```
Or with a simple GUI by just running the jar file:
```
java -jar CanvasExpand.jar
```
